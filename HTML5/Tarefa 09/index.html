Texto marcado

podemos criar também um texto marcado usando a tag <MARK>

Texto pequeno

Podemos também colocar textos reduzidos usando a tag <SMALL>

Texto deletado

Podemos marcar um texto como excluido usando a tag <DEL>

Texto inserido

Um texto inserido ganha ênfase usando a tag <INS>

Texto sobrescrito

Um texto sobrescrito permite escrever coisas como X2+2 usando a tag <SUP>

Texto subscrito

Também podemos usar texto subscrito para fazer o H2O usando a tag <SUB>
