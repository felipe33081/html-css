Links locais, externos e download de arquivos

Os links ou hyperlinks são ligações que existem entre páginas. Vamos ver alguns exemplos do uso das âncoras.

Os links locais apontam para documentos que estão no seu próprio site. Acesse outra página neste mesmo servidor clicando no link anterior.

Também podemos colocar documentos em outras pastas locais. Acesse uma página em outra pasta clicando no link anterior.

Também é possível acessar outros sites usando âncoras. Por exemplo, você pode criar um link para acessar ao YouTube (não se esqueça de
clicar no botão "voltar" do seu navegador depois, para voltar para cá).

O link acima faz o usuário ir para outro site e abandona o meu. o ideal é que ele abra outros sites em uma nova aba. Acesse agora o
YouTube em uma nova aba em branco

Os links também podem ser usados para fazer o download de arquivos de outros formatos como ZIP, PDF, RAR, etc. Por exemplo, abra
aqui um arquivo PDF de exemplo ou baixe a versão compactada do livro.

OBS:Clique sempre em exibir o código fonte para ver como tudo está sendo criado.
